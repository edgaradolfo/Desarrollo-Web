Detalle: En este formulario se permite el manejo de archivos. Esto quiere decir que aca se pueden ver todos los archivos que el usuario administrador desee. Estos luego se le mostraran al usuario registrado para que pueda descargarlos y utilizar esas imagenes a la hora de hace su pagina principal. No solo imagenes, tambien se permite subir videos y/o algun otro archivo que sea necesario. Tambien se permite eliminarlos.
Detalle: En este formulario el usuario administrador puede modificar la página principal para los usuarios registrados de manera general. Estos deberan despues entrar acá y cambiarlo a su estilo. Acá se presentan 3 editores wysiwyg que permitirán armar la pagina de una manera mas fácil. También se presentan varias opciones, como ser el título de la página, sus editores css y js para darle estilo a la página y una breve descripción de la página.
Detalle: En este formulario se le permite al administrador elaborar paginas estaticas que los usuarios podran usar cuando esten arman su pagina principal. En este se presenta un editor html en donde ira todo el contenido de la pagina. Tambien se proveen otras opciones como ser la url de la pagina, su pagina padre, si tendra cabezado o no, si tendra pie de pagina o no, la implementacion del breadcrumb entre otros.
Detalle: Es en este formulario donde se elaboraran posts de diferentes tipos. Pueden ser posts informativos, musicales, entre otros. Al usuario administrador se le da la opcion de poner manejar o gestionar lo que son todos los diferentes posts. Todo esto para poder tener un mejor control de todo lo que pasa en la pagina. A la hora de hacer un post se brinda un editor html para el contenido y otras opciones a llenar.
Detalle: En este formulario el usuario administrador tiene un control sobre lo que son los comentarios que han hecho todos los usuarios en diferentes posts, todo esto para poder tener un mejor control de la información. En dicha tabla se filtran los comentarios por usuario, lo que dice el comentario y en que post fue que se insertó el comentario. El acceso esta restringido a solo los administradores, los usuarios registrados no.
Detalle: En el formulario de Creación de Temas es donde los usuarios podrán elegir los diversos temas que se les presentan. Estos temas son alimentados por este formulario al cual tiene acceso el usuario administrador y es el que llenara con imagenes y estilos cada tema a la hora de crearlos. Por cada tema hay opciones a llenar como ser el nombre o título del tema y una breve descripción sobre que trata el tema que fue creado.
