Detalle: En este formulario se permite el manejo de archivos. Esto quiere decir que aca se pueden ver todos los archivos que el usuario administrador desee. Estos luego se le mostraran al usuario registrado para que pueda descargarlos y utilizar esas imagenes a la hora de hace su pagina principal. Detalle: En este formulario el usuario puede modificar la página principal que fue hecha por el Administrador como página por defecto. Estos deberan despues entrar acá y cambiarlo a su estilo. También se presentan varias opciones, como ser el título de la página, sus editores css y js para darle estilo a la página y una breve descripción de la página.
